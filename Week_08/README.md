##### 1、位运算

实战位运算要点

* 判断奇偶：
  x % 2 == 1 —> (x & 1) == 1
  x % 2 == 0 —> (x & 1) == 0
* x >> 1 —> x / 2. 
  即： x = x / 2; —> x = x >> 1;
  mid = (left + right) / 2; —> mid = (left + right) >> 1;
* X = X & (X-1) 清零最低位的 1 • X & -X => 得到最低位的 1 • X & ~X => 0
